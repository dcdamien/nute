Антон Страхов – Кол-во гейтов выясняем после замеров зон покрытия на полигоне.
Гейты отсеков подключены к компьютерам по usb (хаб), компьютеры объединены в сеть по lan. 
Задачи гейтов:
получить id локации от браслета игрока по радио
передавать серверу id локации браслета
передавать браслету события/команды от сервера по радио
передавать серверу id подключенной пилюли
записывать на подключенную пилюлю данные (кол-во зарядов)
