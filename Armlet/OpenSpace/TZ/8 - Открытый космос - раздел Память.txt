Открытый космос - раздел Память
Память - раздел на браслете про элементы сознания типа флэшбек.
В разделе игроку доступны подразделы:
1.  Вскрытые флэшбеки - список, сортировка в порядке вскрытия игроком (самые старые - внизу списка). Прокрутка Х-Y или L/R, если по-странично.  

Список формируется из всех элементов сознания игрока типа "флешбек" и параметрами "вскрытый" и тэг личности (тэг личности фб должен совпадать с текущим тэгом личности игрока)

Любой из списка можно просмотреть (кнопка Х) - открывается текст флешбека. Прокрутка текста Х-Y, переход на следующий/предыдущий вскрытый флешбэк из списка кнопками R/L, возврат в список - Е

2. Закрытые флэшбеки - список доступных в данный момент триггеров-условий для вскрытия флешбека. Прокрутка Х-Y или L/R, если по-странично.  

Список закрытых флэшбеков формируется из всех флэшбеков игрока, которые подходят под условия "закрыт" + выполнены все дополнительные условия 
Дополнительные условия - указан фб (или несколько), который должен быть в состоянии “вскрыт”, чтобы триггер данного фб появился в списке закрытых фб на браслете. Если в условии несколько фб, то - перечисление через запятую = логическое И. логическое ИЛИ прописывается явно.
Метод вскрытия - бывает 3 вида: по-умолчанию - метод не указан, может быть вскрыт игроком и мнемо-хиругом. “только игроком” - фб не может быть вскрыт мнемо-хирургом; “только мнемо-хирургом” - не может быть вскрыт игроком. 
(мнемо-хирург здесь - вскрытие через модель на терминале = только через сервер). 
Действие - дополнительные действия после того, как фб вскрыт.
бывают следующие:
1) сменить текущую личность игрока на “тэг” = все остальные фб и элементы с тэгам других личностей (вскрытые и закрытые) скрываются на браслете, отображаются фб новой личности
2) вернуть фб из вскрытых в закрытый - указанный фб заново закрывается. важно!  - это действие должно произойти последним в обработке
3) запустить болезнь “название” - запускает медицинское событие на браслете игрока

Доступный по условиям фб/триггер из списка игрок может вскрыть - кнопка А. По нажатию - диалоговое окно "Вы уверены, что хотите вскрыть флэшбек "%текст триггера%" ? Z - да, С -нет.
При вскрытие флэшбека выводится его текст на экран, меняется параметр элемента на "вскрытый". Прокрутка текста Х-Y,  возврат в список триггеров - Е

Каждый фб имеет один или несколько тэгов личности "имя".

Все фб, которые не совпадают по тэгу с текущим тэгом личности игрока, должны быть "скрыты" от игрока на браслете. Если у фб несколько тэгов, то достаточно совпадения по одному из.

3. Мои знания и навыки - список элементов сознания игрока. Формируется из тех элементов, тэг личности которых (или один из) совпадает с текущим тегом личности игрока. . Прокрутка Х-Y или L/R, если по-странично.  
Список = названия элементов, принадлежащих зеленому спектру сознания + значимость. 
Значимость - это текстовое описание параметра "объем" элемента, выраженный так:
1-5 несущественный
6-15 небольшое значение
16-25 что-то было
26-50 факт жизни
50-100 имеет значение
101-200 важный 
201-400 трепетное отношение
400+ жизненно необходимый

Любой элемент из списка можно просмотреть (кнопка Х) - открывается текст. Прокрутка текста Х-Y, переход на следующий/предыдущий вскрытый текст из списка кнопками R/L, возврат в список - Е
